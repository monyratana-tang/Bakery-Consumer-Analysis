# Bakery Consumer Insights

## Overview
This project analyzes bakery transaction data using R to understand customer purchasing behavior, product demand, and item associations.

## Methods
- Data Cleaning & Exploration (dplyr, ggplot2)
- Product Demand Analysis
- Time-Based Analysis
- Apriori Association Rule Modeling

## Key Insights
- Coffee is the most purchased item
- Demand is concentrated around a few core products
- Strong associations exist between coffee and breakfast items

## Recommendations
- Bundle coffee with pastries and toast
- Optimize store layout for cross-selling
- Focus promotions on high-demand combinations
